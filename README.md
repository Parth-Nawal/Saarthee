# Saarthee

Saarthee is a simple desktop voice assistant made using Python.
It listens to voice commands and performs basic tasks like opening websites, telling the time, and running system applications.

This project was built mainly for learning and practice, focusing on Python automation and voice recognition.

## What it can do
- Take voice input through the microphone
- Convert speech to text
- Reply using voice output
- Open websites and applications
- Tell current time and date

## Built With
- Python
- speech_recognition
- pyttsx3
- datetime
- webbrowser
- os
- pyautogui

## How to run
Install dependencies:
pip install SpeechRecognition pyttsx3 pyautogui

Run the program:
python saarthee.py
